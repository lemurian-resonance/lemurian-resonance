---

### 🦁 レムリア交信（Git反映）の3ステップ

**1. 変更をすべて集める**
```bash
git add .
```
> **意味:** 「今書き換えたすべてのファイル（.htmlや画像など）を、次に送るリストに登録するガオ！」

**2. 記録に名前をつける**
```bash
git commit -m "update"
```
> **意味:** 「この更新に『アップデートしたよ』というラベルを貼って、記録を確定させるガオ。-m の後の引用符の中は、自分がわかりやすい名前に変えてもいいガオ。」

**3. 世界（GitHub）へ送り出す**
```bash
git push
```
> **意味:** 「これで、インターネット上のレムリアに私の変更が届いて、みんなが見られるようになるガオ！」

---

### 🦁 もし、途中で「今どうなってる？」と思ったら
```bash
git status
```
> **意味:** 「どのファイルが変更されて、まだ登録（add）されていないかを確認できるガオ。迷った時はこれを打つと安心だガオ。」
